# HYBRID-RAG-CAPSTONE

Hybrid RAG capstone project — retrieval-augmented generation pipelines and evaluation utilities.

Quick start

1. Create and activate a virtual environment:

```powershell
python -m venv .venv
& .venv\Scripts\Activate.ps1
```

2. Install dependencies:

```powershell
pip install -r requirements.txt
```

3. Run tests:

```powershell
pytest -q
```

Project layout

- `main.py`, `backbone.py`, `test_llm.py`
- `src/` — package modules for pipelines, analyzer, utils, and evaluation
